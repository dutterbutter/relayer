Overview

Services are tasks which the relayer constantly runs throughout its lifetime. Services handle keeping up to date with the configured chains.

Sets up the web socket server for the relayer, routing (endpoint queries / requests mapped to handled code) and instantiates the database store. Allows clients to interact with the relayer.

Starts all background services for all chains configured in the config file.
